# Week 1 - Python Foundations

## Goal of Week 1

The goal of this week is not to become expert.  
The goal is to become comfortable opening VS Code, writing simple Python programs, running them, fixing small mistakes, and committing work to GitHub.

## Topics Covered

- print()
- comments
- variables
- basic data types
- user input
- simple mini project

## Week 1 Success Criteria

By the end of Week 1, I should be able to:

- Create and run a Python file
- Use print statements
- Write comments
- Store values in variables
- Understand text, numbers, and basic data types
- Take simple user input
- Push code to GitHub
